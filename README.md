# VE ABSOLUTA - Módulo de Ingesta (Frontend)

Interfaz de usuario para el sistema **VE ABSOLUTA**. Este módulo permite al usuario final cargar archivos multimedia y visualizar los veredictos emitidos por el motor de detección en tiempo real.

## 🛠️ Stack Técnico
* **Librería Principal:** React.js.
* **Comunicación:** Axios para peticiones asíncronas al Backend.
* **Estado:** React Hooks (useState, useEffect).

## 🎨 Estado del Proyecto (UI/UX)
La lógica de conexión con el Módulo de Orquestación ya está implementada y operativa. El diseño visual se encuentra actualmente en fase de prototipado y mejora estética.

## 🚀 Instalación
1. Clonar repositorio.
2. Instalar dependencias: `npm install`.
3. Iniciar entorno de desarrollo: `npm start`.

---
*Nota: Asegúrese de tener el backend (puerto 8080) corriendo para que la carga de imágenes funcione correctamente.*
